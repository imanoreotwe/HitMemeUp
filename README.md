# HitMemeUp
A flask based, docker contained, text based, meme searcher. 
To try it out text any meme query to +17205130277 and recieve a matching meme.
`docker-compose` file is there for you to run on your own hardware.

# How to use
1. Text 17205130277
2. chose format
  * meme <query>
  * gif <query>
  * insult
3. enjoy
